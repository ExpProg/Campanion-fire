# **App Name**: Campanion

## Core Features:

- Landing Page: Display a landing page with clear calls to action for registration and login.
- Camp Listing: Display a list of camps with their basic information (name, description, dates, location, main image, and price).

## Style Guidelines:

- Use a clean, single-column layout to focus attention.
- Use a neutral color palette as a background with a single, bright accent color to draw attention to important elements. The color palette is: 05668d, 028090, 00a896, 02c39a, f0f3bd
- Accent color: Use a vibrant Teal (#02c39a) for buttons and interactive elements.
- Use a modern sans-serif font for all text elements.
- Employ a set of simple, consistent icons throughout the application.
